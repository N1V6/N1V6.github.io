# 为什么要写周记

有句话说：人不能同时拥有青春和对青春的感觉。我觉得很有道理，不只是青春，人生每个阶段都有不同的经历，即使对同一段经历也有不同的感受。我很喜欢翻之前学生时期写的随笔，感慨当时只道是寻常，现在我把自己的碎碎念记录下来，过几年后再看应该也会觉得很有意思。

为什么不选择周记或月记，一是因为自己的生活和想法没有能够到每天去更新一次的丰富程度，一旦断更了，那就完了，出现一次就会有无数次断更；月记的时间跨度太长了，我笨头脑，容易忘事。

这是《南海十三郎》江誉镠的一段台词，我当然不像他志向远大去实现 “文章有价”，只是觉得这种理想很美好很有力量，如果离开前能留下一点痕迹，也不会觉得白活。

静态部署 github 上，还是比较稳妥的，不用买主机和域名，就是墙内访问不稳定，页面自由度也没那么大，不过也够用了，内容都存在本地，想啥时候转移到主机上再说。

# 补完《权力的游戏》

之前看了前四季，刚好这周出差，下班后无聊捡回来看，花了一周基本补完了。第五六季很不错，每条支线都让人有想看下去的欲望，各个角色也在一直成长，没有过多无用的感情戏，S6E9 的私生子之战，snow 经典名场面；S6E10 开场的剧情配合恰到好处的神 BGM 也看得很爽。

但到七八季有点拉了，小恶魔原本那么有谋略的人设反而成了自家阵营的卧底，打夜王直接送龙也看得血压高，龙妈后期阵营这么强大，编剧非得去硬整骚操作硬削弱，第七季看到一半就直接去看剧情梗概了。

# 《海上钢琴师》重映

最近重映的电影还挺多，上上周重温了《这个杀手不太冷》，这周五《海上钢琴师》又重映，评价还挺高，我还是比较期待的。

但看完会我感觉一般，甚至途中还刷了会手机。可能是内心比较烦躁，我找不到共鸣点，情节也略感夸张和刻意，我所理解的这部电影想表达的意思是，不想让世俗的纷扰干扰自己安宁纯净的丰富的内心世界？ 斗琴的情节还不错，但剧情总体平平淡淡，没留下什么深刻的印象。当然这些都是主观想法，可能是我没看懂。

# 关于朋友圈

和朋友聊到一个话题：发朋友圈主要是给以后的自己看的还是给别人看的？

我的观点偏向于后者。朋友圈本身就有社交属性，无论是发发牢骚、表达观点还是分享美好生活，个人觉得主要目的都是希望和好友创造社交联结，这些社交联结主要体现在点赞，评论，以及好友对你的看法，这些他人的评价与认同，都是对自我情感体验的正向反馈，发朋友圈当然没毛病，我也发，我还喜欢在发完的半小时内刷刷谁给我点赞评论了。

“现实中一地鸡毛，朋友圈岁月静好”。很多人去旅行，明明累的要死，还要趁 ip 还没改回来，拖着累垮的身体从几百张摆拍中挑出九张来高 P，再配着从小红书精挑细选来的文案发到朋友圈，回复底下的评论还需要字斟句酌生怕掉了 b 格，这有毛病吗？个人觉得没有（只要不过于虚伪），塑造一个 “理想化” 的自我来获得认同需求，本身就是朋友圈的意义。

题外话

“生活乏趣无味，牛马点评人类”

有时候喜欢去翻不熟的人的朋友圈，看他们发的内容，再去揣测他是一个怎样的人。这种幻想自己站在 “上帝视角” 的吊毛事我以前也不少干：大学上心理课的时候，特地坐在最后一排，当老师说出一些犀利的观点或者羞耻的论题时，我喜欢去观察同学的反应，非常有趣，乐此不疲。

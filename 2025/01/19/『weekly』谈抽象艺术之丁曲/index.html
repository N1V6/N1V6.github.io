# 谈抽象艺术之丁曲

# 什么是丁曲

但丁的《神曲》被誉为意大利文学史上的奇迹，文艺复兴时代人文主义的曙光，但是，我对它一点兴趣也没有，我想聊的是另一个在 20 世纪抽象艺术史上具有里程碑意义的巅峰之作 —— 丁真的二创歌曲

# 从 202 1！ 到 202 5！

“丁曲” 一般是指视频创作者基于丁真梗文化对一些歌曲进行重新填词、调音等操作后，创作出的二创歌曲。

rap 在众多音乐风格中，更注重对创作者自身的表达，所以歌词有时候显得比旋律更重要，而对于丁曲而言，创作者刚好需要通过歌词来表达对丁真文化现象的调侃、嘲讽或其他更复杂的情感，同时，rap 形式的歌曲制作门槛相对较低，又具有强烈的节奏性和律动感，十分上头洗脑，所以早期的丁曲多以 rap 的形式发行。

比如 2021 年横空问世的《zood》，作为” 丁曲 “的第一笔横，虽不是开山之作，却正式宣告丁曲势力的崛起，也奠定了早期丁曲的风格基调。深受《zood》的影响，更多的 rap 风的丁曲接踵而至，《stay》remix、《I Got Smoke》、《烟 distance》等作品更加扩大了丁曲的影响力，开启了丁曲的大制作时代。

这些作品的成功，启发了更多创作者投身丁曲创作。慢慢地，大家也不再满足于最初的 rap 风格，开始大胆尝试各类音乐形式。比如古风丁曲的代表作《丁花词》，饱含诗意，雅俗结合，为丁曲增添了浓厚的文化底蕴，论文学创作性，又不得不提到《老街》，歌词采用顶针的修辞手法的同时又不失内涵；《肺痒痒》、《爸爸去哪儿抽》作为儿歌丁曲的代表，旋律简单易记，更加上头和洗脑；也有戏剧风格的作品，比如《丁家坡》；民谣经典《理塘烟馆》；摇滚标志性巨作《波西米亚狂想抽》；流行歌曲《群丁》、《一了百了》等等......

# 都什么年代了，还在听传统丁曲

丁曲往往不直接在歌词中提到丁真，而是靠着一些相关的意象来借物喻人，比如理塘、纯真、电子烟（肺）、我测尼玛、小马珍珠、雪豹、国企（编制）、小镇做题家、阿玛等。早期的丁曲主要围绕丁真的 “电子烟”、“纯真” 来创作，带有讽刺、嘲笑甚至谩骂的意味，内容较为直白和攻击性，多流于表面，没有对社会现象等方面进行深度挖掘。

但随着发展，大伙不再仅仅局限于对丁真个人形象的简单调侃和攻击，而是开始将目光投向更为广阔的社会层面，逐渐丰富歌词的内涵，深入探讨各种社会问题和人生百态。越来越多的意象涌现出来，叙事风格和角度也在变化。最有叙事代表性的作品是《理塘烟馆》，以舅舅为主角，倒叙插叙了多段情节，从另一个角度对丁真现象进行解构。还有利用谐音进行双重填词的《大石测胸口》，在有限的文字空间内，融入了更深层次的表达。

在如此众多叙事风格中，“if” 线是最有艺术代表性的，在这个时间线里，丁真没有现实生活中的鸿天齐云蛊，仅是像我们一样的普通人。

《丁家坡》里的丁真心中有志，步入社会却被屡遭挫折，与阿玛对话，心中有苦却诉不得，欲说还休，唯不让母担忧，而另一个平行时空身披彩衣、脚踩红地毯的自己仅是梦中场景，不禁潸然泪下。在众多丁曲中，讽刺题材常见，衍生意象也颇多，但《丁家坡》能从另一个维度出发，结合现实、乡愁、不公等要素，融入到底层劳动人民的生活里，引起老百姓的强烈共鸣，实乃艺术，在丁曲排行里毫无疑问是 top3 的存在。

《曾经我也想过一了百了》也是如此，作者在演唱时抛弃了先前的 “丁腔”，寓意着这个平行时空的丁真已经学会了流利的普通话，离开了家乡理塘，步入社会，从一个更加宏大的叙事角度，展示了众生百态。

if 线的丁曲，看似是在唱丁真，实际是在叙说大道上的芸芸众生。

# 唐诗宋词 “丁曲”

丁曲非常类似于古代的词牌名。在唐宋时期，文人们会根据已有的格律来填写新的诗词，形成不同内容和风格的词作，比如《江城子》《满江红》等词牌名都有众多文人墨客创作出不同版本。而丁曲通常也是采用已有的音乐旋律，对歌词重新创作，配上纯真的理塘口音再加工而来的。

更加有趣的是，在内容情感表达的发展过程上，它们也有相似之处。

比如唐初诗词多描写游玩之喜、抒发豪情壮志或对山水景观的赞叹，在诗人步入官场或经历波折后，所创作的诗词更多的是对社会现实的批判，对人民苦难的同情和对国家命运的担忧。

早期的丁曲多侧重于对丁真本人的攻击和嘲讽，玩梗和泄愤的成分居多，但随着时间的推移，不管大众是经过了文化的熏陶还是现实的洗礼，越来越多的人开始对丁真 “神” 的身份进行结构，发现丁真也不过是一颗被无形大手肆意拨弄的棋子，这场造神运动的始作俑者更是碰都不能碰的话题，消灭了这个丁真，还会有下一个甲真、乙真坐到这个位置上。

于是，从这个时候开始，丁曲逐渐延申到探讨社会现象、人生价值等更为广泛的话题，从最初的小众网络亚文化产物，逐渐成长为一种能够反映社会现实、表达大众情感的独特艺术形式。

而在这些丁曲里，最有代表性的无疑是被誉为 “孤篇压全唐” 的《群丁》

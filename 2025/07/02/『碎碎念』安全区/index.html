# 安全区

要想得到一个人内心深处的秘密，你必须先将自己心底同等分量的秘密交付于他。

倘若你所求的那个秘密，是羞耻的、污秽的、难以启齿的，那么就得委屈自己 “同流合污”。不管是伪装也好，违背本心也罢，为了最终的目的，必须毫无保留地释放你的同理心，与他感同身受，用自己编造的同样不堪的 “亲身经历”，去层层瓦解他的戒备。

在这样的环境下，每个人身上都沾有污泥，没有鄙夷的目光，也没有道德的审判。那些所谓道德败坏之事，开始变得理所当然，一切都得到了合理的解释。也只有在这样被默许的、安全的黑暗中，人们才敢于真正地，袒露自己内心的阴暗。

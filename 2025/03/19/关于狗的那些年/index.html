关于狗的那些年

发表于 2025-03-19
  • N1V
首页

# 一、

“要想从此过，留下买路财”，这是师傅大黄给小白的第一次试炼，奈何运气不佳，他不知眼前的这位人类，是何等的绝世高手......

徒弟败阵，师傅出马。“下山之后，切莫轻敌”，三年前世尊的叮嘱大黄不曾忘记，只是此战之前，未曾出现过能让他睁开双眼的挑战者。

如今，面对此等强敌，是该破例 —— 睁开一只眼了。

# 二、

默，这个名字曾在江湖中如雷贯耳，十几年前，他凭借着独门秘技 “薄日掌火”，在江湖中纵横驰骋，所到之处，血雨腥风。

那年武林大会，豪杰毕至，他闲庭信步，力挫群雄，夺得盟主之位，本是功成名就之时，他却忽然绝迹江湖，无狗知其所踪。

再见时，他已是这片菜园的一条看门狗，数十年江湖梦，只说与黄花听。

# 三、

税重如山，寝食难安，灾荒连年，苦不堪言。即便昨夜老五冻死在屋内，家中也还有四张小嘴待填，嗷嗷凄叫，揪着大狗的心。

老大高不过路边杂草，也要跟着大狗外出求生，于山中寻野果，至河泽探鱼虾。

昨日痛失亲骨肉，明日还需愁生计，死神追随他的脚步，嗅探他的踪迹，却迟迟不挥斩手中的镰刀。

这是对他的慈悲，还是惩罚。

# 四、

在你面前的，是阿咪家紧闭的大门。今早你就听闻阿咪家来了只新猫。

和往常一样，你站在门口等她出来，但直至夕阳染红了天空，也不见她的身影。流浪一生教会了你如何分辨危险的气味，但没人告诉你，心动的感觉像是吞了颗会发芽的春药，刺激苯基乙胺不断分泌。

在你身后的，是拿着手机对着你傻笑的人类。临走前，他给你留下一句话：

"猫狗殊途，你舌尖的倒刺，永远理不顺她的绒毛，何不吞下呜咽，像后面这只狗一样，孤身浪迹天涯呢？"

# 五、

后面这只狗，上辈子不知做了什么孽，今世降临狗间那刻，上天就剥夺了他爱与被爱的权利。

之所以说是上天所为，那是有证据可循的，这只狗左胸口的伤痕，正是被老天爷一拳打落凡间所致。

出生时没有收到命运的请柬，却自带三根反骨 —— 第一根对着天空竖起中指，第二根把每道车辙都走成岔路，第三根将路边的广告牌挠成抽象派。

现在，他估计是想用矫健的四条腿逼迫世界掏出藏好的肉骨头，又或许是在用爪痕盖上自由纹章向律法证明他曾经来过。

谁知道呢。

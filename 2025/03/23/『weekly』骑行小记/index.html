# 骑行小记

相比于平坦的马路，我更向往这些未曾被踏足的山中小径，它们往往藏着惊喜

前有绝景，数数看有几只猫猫

看向镜头的狗

钓鱼佬的天堂

山中发现一所小庙，往前触发支线剧情

回去路上，猫猫还在那坐着，不知道在等谁
